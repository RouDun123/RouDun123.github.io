如何在github上部署Hexo博客

搭建博客

1下载git客户端

下载git

安装gitBash

注册github账户

2配置git

右键一个空文件夹（放博客本地仓库的文件夹）点git bashhere

gitBash添加全局用户名密码

进入/c/Users/xxx/.ssh

使用cd命令进入密钥存储文件

生成本地ssh-key

复制.pub后缀的公钥

github.com/你的用户名

点击你的头像 -> setting -> 左侧栏选SSH and GPG Keys -> new sshKey

把刚才的公钥复制到key里面，Title随便取

测试连接

GitHub 主页右上角加号 -> New repository：

  • Repository name 中输入 用户名.github.io
  • 勾选 “Initialize this repository with a README”
  • Description 选填

填好后点击 Create repository 创建。

3创建 git page hub

GitHub 主页右上角加号 -> New repository：

  • Repository name 中输入 用户名.github.io
  • 勾选 “Initialize this repository with a README”
  • Description 选填

填好后点击 Create repository 创建。

4安装HEXO

先安装npm，去nodejs官网下载

访问 http://localhost:4000，出现 Hexo 默认页面，本地博客安装成功！
